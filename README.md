# Wuhan 2020 React Native App

## development

* `yarn start` => start bundler
* `yarn ios` => start iOS simulator
* `yarn android` => start iOS simulator

## production

### Android

`cd android && ./gradlew assembleRelease` and apk in `app/build/outputs/apk/release/`


### iOS

use Xcode
